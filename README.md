# Notes-Web-App

## Setup & Installation

Check to see if you have the latest version of Python installed.

```bash
python --version
```
Clone the repo.
```bash
git clone <repo-url>
```
Install requirements
```bash
pip install -r requirements.txt
```
## Running The App

```bash
python main.py
```
## Viewing The App
Go to `http://127.0.0.1:5000`
